fpdetective
===========

A framework for conducting large scale web privacy studies.

## Get Started

* Check out recent binary [releases](https://github.com/fpdetective/phantomjs/releases).
* [Build modified PhantomJS from Source](https://github.com/fpdetective/phantomjs/wiki/Building-from-source).
* Consult [PhantomJS wiki](https://github.com/ariya/phantomjs/wiki) to learn more about PhantomJS

### Patches for Chromium & PhantomJS browser
You can use following patches to build FPDetective browsers from source. Consult readme files for further explanation.
 
* Chromium: [patch](https://github.com/fpdetective/fpdetective/blob/master/patches/chromium.patch) | [readme](https://github.com/fpdetective/fpdetective/blob/master/patches/chromium-readme.txt)
* PhantomJS: [patch](https://github.com/fpdetective/fpdetective/blob/master/patches/phantomjs.patch) | [readme](https://github.com/fpdetective/fpdetective/blob/master/patches/phantomjs-readme.txt)
